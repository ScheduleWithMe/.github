# 팀원들끼리 시간을 맞춰보자!

## 서비스 플로우

- 회원가입
- 로그인
- 시간을 정하고 싶으면 Time Meet(우리 기능) Create - 주최자
    - Create할 때 정해야 할것
        - 객체 이름 (ex: 8조 회식 시간)
        - 객체 주간 (ex: 4월 17일 ~ 4월23일)
        - 객체 장소 (ex: BHC치킨집)
        - 객체 설명 (ex : 회식할거니까 다들 되는 시간 고르세요!)
        - 객체 필요 인원(ex: 6)
        - 위에걸 다 채우고 Create
